# Security Policy

This repository is a portfolio project and does not use live pricing engines, customer records, or production billing data.

## Reporting

If you identify a security issue in the project shell or development workflow, report it through GitHub Security Advisories or via the official contact routes on [Kinetic Gain](https://kineticgain.com/).

## Scope

- Demo data remains synthetic.
- No commercial secrets or pricing engine credentials should be committed.
- Package hygiene should continue through Dependabot and CI.
